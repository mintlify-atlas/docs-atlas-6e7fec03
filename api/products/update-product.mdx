---
title: "Update Product"
api: "PUT /api/productos/sku/{sku}"
description: "Update an existing product by its SKU (Admin only)"
---

## Authentication

<Warning>
  This endpoint requires administrator authentication. Include a valid JWT token in the Authorization header.
</Warning>

<ParamField header="Authorization" type="string" required>
  Bearer token for admin authentication
  
  Example: `Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`
</ParamField>

## Path Parameters

<ParamField path="sku" type="string" required>
  The Stock Keeping Unit (SKU) of the product to update
</ParamField>

## Request Body

All fields are optional. Only include the fields you want to update.

<ParamField body="nombre" type="string">
  Product name (max 155 characters)
</ParamField>

<ParamField body="sku" type="string">
  New SKU code (max 50 characters, must be unique)
</ParamField>

<ParamField body="descripcion" type="string">
  Detailed product description
</ParamField>

<ParamField body="precioCantidad" type="decimal">
  Price amount
</ParamField>

<ParamField body="precioMoneda" type="string">
  Currency code (3 characters)
</ParamField>

<ParamField body="dimensionesAlto" type="double">
  Product height dimension
</ParamField>

<ParamField body="dimensionesAncho" type="double">
  Product width dimension
</ParamField>

<ParamField body="dimensionesProfundo" type="double">
  Product depth dimension
</ParamField>

<ParamField body="es_destacado" type="boolean">
  Whether the product should be featured
</ParamField>

<ParamField body="stock" type="integer">
  Available quantity in inventory
</ParamField>

<ParamField body="imagen_url" type="string">
  URL to product image
</ParamField>

<ParamField body="categoria" type="object">
  Category information
  
  <Expandable title="Category Object">
    <ParamField body="categoria_id" type="long">
      Category unique identifier
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```bash cURL
curl --location --request PUT 'https://api.iquea.com/api/productos/sku/IKEA-TABLE-001' \
--header 'Authorization: Bearer YOUR_ADMIN_TOKEN' \
--header 'Content-Type: application/json' \
--data '{
  "precioCantidad": 279.99,
  "stock": 20,
  "es_destacado": false
}'
```

```javascript JavaScript
const token = 'YOUR_ADMIN_TOKEN';
const sku = 'IKEA-TABLE-001';

fetch(`https://api.iquea.com/api/productos/sku/${sku}`, {
  method: 'PUT',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    precioCantidad: 279.99,
    stock: 20,
    es_destacado: false
  })
})
.then(response => response.json())
.then(data => console.log(data));
```

```python Python
import requests

token = 'YOUR_ADMIN_TOKEN'
sku = 'IKEA-TABLE-001'
url = f'https://api.iquea.com/api/productos/sku/{sku}'

headers = {
    'Authorization': f'Bearer {token}',
    'Content-Type': 'application/json'
}

data = {
    'precioCantidad': 279.99,
    'stock': 20,
    'es_destacado': False
}

response = requests.put(url, json=data, headers=headers)
product = response.json()
```

  Product successfully updated
</ResponseField>

</ResponseField>
