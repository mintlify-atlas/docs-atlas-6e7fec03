---
title: "Get Product by ID"
api: "GET /api/productos/{id}"
description: "Retrieve a single product by its unique identifier"
---

## Authentication

This endpoint is publicly accessible and does not require authentication.

## Path Parameters

<ParamField path="id" type="long" required>
  The unique identifier of the product to retrieve
</ParamField>

## Response

Returns a single product object with complete details.

<ResponseField name="producto_id" type="long">
  Unique identifier for the product
</ResponseField>

<ResponseField name="sku" type="string">
  Stock Keeping Unit - unique product code
</ResponseField>

<ResponseField name="nombre" type="string">
  Product name
</ResponseField>

<ResponseField name="descripcion" type="string">
  Detailed product description
</ResponseField>

<ResponseField name="precioCantidad" type="decimal">
  Price amount
</ResponseField>

<ResponseField name="precioMoneda" type="string">
  Currency code (e.g., "USD", "EUR")
</ResponseField>

<ResponseField name="dimensionesAlto" type="double">
  Product height dimension
</ResponseField>

<ResponseField name="dimensionesAncho" type="double">
  Product width dimension
</ResponseField>

<ResponseField name="dimensionesProfundo" type="double">
  Product depth dimension
</ResponseField>

<ResponseField name="es_destacado" type="boolean">
  Whether the product is featured
</ResponseField>

<ResponseField name="stock" type="integer">
  Available quantity in inventory
</ResponseField>

<ResponseField name="imagen_url" type="string">
  URL to product image
</ResponseField>

<ResponseField name="categoria" type="object">
  Category information
  
  <Expandable title="Category Object">
    <ResponseField name="categoria_id" type="long">
      Category unique identifier
    </ResponseField>
    
    <ResponseField name="nombre" type="string">
      Category name
    </ResponseField>
    
    <ResponseField name="slug" type="string">
      URL-friendly category identifier
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl --location 'https://api.iquea.com/api/productos/1'
```

```javascript JavaScript
fetch('https://api.iquea.com/api/productos/1')
  .then(response => response.json())
  .then(data => console.log(data));
```

```python Python
import requests

product_id = 1
response = requests.get(f'https://api.iquea.com/api/productos/{product_id}')
product = response.json()
```
</RequestExample>

## Status Codes

<ResponseField name="200" type="OK">
  Successfully retrieved the product
</ResponseField>

<ResponseField name="404" type="Not Found">
  Product with the specified ID does not exist
</ResponseField>
